# Ảnh cho website Ohara Flora

Bỏ ảnh hoa của shop vào ngay thư mục NÀY (D:\Coding\images), đặt đúng tên file bên dưới.
Không cần sửa code. Trang sẽ tự lấy ảnh của bạn. Nếu thiếu file nào,
trang tạm hiển thị ảnh mẫu thay thế nên không bị vỡ layout.

Định dạng nên dùng: .jpg (hoặc đổi đuôi trong file HTML nếu bạn dùng .png/.webp)

------------------------------------------------------------
TÊN FILE  ->  DÙNG Ở ĐÂU  ->  KÍCH THƯỚC ĐỀ XUẤT (rộng x cao)
------------------------------------------------------------

HERO - CUNG TRÒN ẢNH (12 ảnh, quan trọng nhất):
  arc-1.jpg ... arc-12.jpg   Vòng cung ảnh ở đầu trang     600 x 600 (VUÔNG 1:1)
  - Cần đúng 12 tấm, crop vuông. Đây là ấn tượng đầu tiên của khách.
  - Chọn 12 tấm hoa/bó hoa đẹp, nhiều màu để vòng cung sinh động.
  - Thiếu tấm nào trang sẽ tạm hiện ảnh mẫu nên không bị vỡ.

  - Video tự phát, đã tắt tiếng và lặp lại (bắt buộc tắt tiếng để trình duyệt cho tự phát).

MẸO:
- Ảnh vuông thì crop vuông, ảnh dọc thì crop dọc -> nhìn cân đối nhất.
- Nén ảnh dưới ~400KB/ảnh để web tải nhanh (dùng tinypng.com hoặc squoosh.app).
- Nếu file của bạn là .png hay .webp: cứ báo mình, mình đổi đuôi trong code cho.
