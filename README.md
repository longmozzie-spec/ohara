# Ohara Flora — Sydney Floral Studio

Landing page for Ohara Flora, a Sydney florist.
Tagline: "Crafting moments, blooming love."

## Run locally

It is a single static file. Open `ohara-flora.html` in a browser, or serve the folder:

```bash
npx http-server -p 5500 -c-1
```

Then visit http://localhost:5500/ohara-flora.html

## Structure

- `ohara-flora.html` — the whole site (HTML + Tailwind via CDN + vanilla JS)
- `images/` — bouquet photos and customer videos

Fonts: Cormorant Garamond (headings), Lato (sub-headings), Open Sans (body).
